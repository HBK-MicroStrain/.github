# MicroStrain by HBK
Software tools for MicroStrain sensors.

---

## Tools

- **[MIP SDK](https://github.com/HBK-MicroStrain/mip_sdk)** - C/C++ SDK for communicating with MicroStrain inertial sensors, with support for bare-metal embedded systems.
- **[MSCL](https://github.com/HBK-MicroStrain/MSCL)** - C++ API with multi-language bindings for communicating with MicroStrain wireless sensors.
- **[Magnetometer Calibration](https://github.com/HBK-MicroStrain/microstrain_mag_cal)** - Calibration tool for MicroStrain inertial sensors.
